# SelfHosted Lab 🧪

自托管基础设施学习笔记和实践记录。

## 📋 内容

### 网络诊断 `networking/`
- 延迟、路由、丢包等基础网络诊断
- VPS 网络质量分析
- Ping/Traceroute/MTR 工具使用

### AI Agent 学习 `docs/`
- AI Agent 工作流探索
- VPS 部署和自动化
- 网络诊断与优化实验

### 配置模板 `configs/`
- 配置文件示例（敏感信息已脱敏）

### 报告分析 `reports/`
- 网络测试报告汇总
- 性能分析样本
